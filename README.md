cbabos.yourweb.hu
=================

This is the beginning of developing my personal, and work-related blog. 
It's created using the following tools: 
 - Node.js
 - gulp
 - Bootstrap (TWBS)

MVP features: 
 - posts written in markdown
 - build script to create html from markdown
 - js, css, html linter
 - css3 autoprefixer, builder
 - js builder
 - html builder
 - version bump
